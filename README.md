## 매일 백준 코테 스터디
### 스터디 기본 규칙
- 주 6회 1일 1문제 풀기
- 불가피한 일정이 있을 경우 팀원 전원 동의 하에 면제 가능
- 벌금은 피보나치 수열로 증가

### 문제 풀 때 규칙
- 코드 작성하는 중간에 절대 실행하지 않는다. 실행은 제출하기 직전 딱 한번만
- 만약 한 번의 기회로 풀지 못했다면, 틀린 코드를 손으로 적고, 왜 틀렸는지 적는다.
- 그리고 다시 문제를 해결하고, 정답 코드도 작성한다.
- 이렇게 작성한 오답노트는 들고다니면서 시간날 때 마다 본다.
